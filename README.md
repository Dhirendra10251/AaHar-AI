# AaharAI — AI-Powered Diet & Wellness for Indian Households

**AaharAI** is an intelligent nutritional platform designed specifically for the Indian context. It bridges the gap between traditional Ayurvedic wisdom and modern AI to provide personalized dietary recommendations based on symptoms, using ingredients commonly found in Indian kitchens.

---

## 🌟 Overview

In many Indian households, health issues like fatigue, acidity, or hair fall are often addressed with home remedies (*Gharelu Nuskhe*). AaharAI digitizes and scales this wisdom by analyzing user symptoms to identify potential nutritional gaps and providing a structured, culturally relevant 7-day meal plan.

### Key Value Propositions:
* **Cultural Relevance:** Focuses on regional Indian diets (from Ragi and Moringa to Methi and Karela).
* **Symptom-Driven:** Identifies potential vitamin/mineral deficiencies based on user-reported symptoms.
* **Accessible Language:** Supports interactions in plain Hindi and English.
* **Practical Guidance:** Provides plans using items available at the local *Sabzi Mandi*.

---

## 🚀 Features

### 1. Symptom-to-Diet AI
Users can describe how they feel—whether it’s "feeling tired after lunch" or "hair fall during exams." The AI cross-references these symptoms with a nutritional database to suggest corrective foods.

### 2. Comprehensive Indian Food Database
A curated library of over **2,000+ Indian foods** mapped to their:
* Nutritional profiles (Vitamins, Minerals, Macros).
* Ayurvedic properties (Vata, Pitta, Kapha balancing).

### 3. Personalized 7-Day Meal Planner
Generates a complete weekly schedule including:
* Breakfast, Lunch, and Dinner.
* Regional alternatives (e.g., swapping a North Indian recommendation for a South Indian equivalent).

### 4. Daily Wellness Nudges
Automated reminders for healthy habits, such as drinking *Jeera* water, eating at consistent times, or adding immunity boosters to meals.

---

## 🛠️ How It Works

1.  **Describe Symptoms:** Input health concerns in natural language.
2.  **AI Analysis:** The engine identifies likely nutrient deficiencies (e.g., Iron, B12, or Fiber).
3.  **Meal Generation:** The system outputs a 7-day plan optimized for those specific nutrients.

---

## 💻 Tech Stack

* **Frontend:** HTML5, Tailwind CSS, JavaScript
* **Backend:** Python (Flask/FastAPI)
* **AI Engine:** Powered by Google Gemini (AMD Promptathon Entry)
* **Deployment:** Google Cloud Run
